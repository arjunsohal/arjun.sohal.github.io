<!DOCTYPE html>

The <!DOCTYPE html> tag is the very first thing put in an html website and it is used to tell the web browser that the website is an html one.

<html>

The <html> tag tells the browser that the document is an html document and the <html> tag also contains the all the contents in the website/document except for the <!DOCTYPE html> tag

<head>

The <head> tag contains all the head elements in the website/document. Some tags the <head> tag contains are the <style> <title> tags but it can also hold other tags aswell.

<style>

The <style> tag defines style information and it specifies how html elements should render in a browser.

<title>

The <title> tag defines a title in the browser toolbar and it displays a title for the page in search-engine results.
