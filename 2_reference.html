<body>

The <body> tag defines the document's body and it contains all of the contents in an html document such as text, tables, lists, images, e.t.c.

<p>

The <p> tag defines a paragraph in html.

<h1> to <h6>

The <h1> to <h6> tags define html headings. <h1> defines the most important heading and <h6> is the least important heading.

<img>

The <img> tag defines an image in html and it has two required attributes which are src and alt. This tag does not need a closing tag

<a>

The <a> tag defines a hyperlink in html, which is used to link pages. The <a> tag needs the attribute href, href indicates the link's destination.
