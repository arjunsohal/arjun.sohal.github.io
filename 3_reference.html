<ol>

The <ol> tag defines an ordered list in html and the <li> tags in between the <ol> tag defines the list's items.

<ul>

The <ul> tag defines an unordered list in html and the <li> tags in between the <ul> tag defines the list's items.

<table>

The <table> tag defines a table in html and it contains the one or more of the following elements <th>, <tr>, or <td>.

<header>

The <header> tag is a container for introductory content in a document.

<footer>

The <footer> tag defines a footer for a document or section.
