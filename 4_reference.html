<!--...-->

The <!--...--> tag is used to insert comments in the source code. The comments don't appear on the actual website and the comments can be used to explain code and that can be useful if you need to come back and make changes to any code.

<div>

The <div> tag defines a section or a division in an html document.

<br>

The <br> tag inserts a single line break and it does not need a closing tag.

<hr>

The <hr> tag defines a thematic break in html and it is used to seperate content.

<b>

The <b> tag specifies bold text.
